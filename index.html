A QUICK GUIDE TO PROGRAMMING

Getting started

Have you ever wondered how computer programs work? Have you ever felt the crawling need to know the secrets behind programming? Then you are in the right place!

What language should I learn?

I suggest to start with an user-friendly language like Python if your goal is to master the language without going to deep in the low-level concepts of programming.

If you are looking for a challenge or you just want to enter the rabbit-hole of programming and really grasp the ideas behind the surface then C (or even better, C++) could be a great choice. C and C++ are indeed incredibly flexible language that gives to the programmer (you!) the ultimate control over the program he's creating.

Java is another good point where to start from. But I personally wouldn't recommend to study Java without any previous knownledge

What is programming all about?

Some may say programming is like writing a letter. Indeed, most of the time dedicated to the process that we call programming is spent writing.

But programming is much more. "To program" means to create, to resolve problems, to experiment, to try, fail and start all over again and again. Programming is a skill you acquire by experience.

Another fundamental concept in programming are heuristics. Just like this one: think of speaking of programming without any actual coding just like speaking of music without an instrument - do you think it's to teach to play the guitar without a guitar? I don't think so, let's dive in!

std::cout << "I'm already learning to code!" << std::endl;
